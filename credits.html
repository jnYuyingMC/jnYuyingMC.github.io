项目依赖

MDUI v2

Material Design 3 前端框架，基于 Web Components，提供按钮、卡片、对话框、菜单等 UI 组件，是本站所有页面的核心框架。

MIT 全站 官网 GitHub

GLightbox

纯 JavaScript 图片灯箱库，支持响应式缩放、键盘导航和触摸滑动，用于本站图片库的点击放大浏览。

MIT index.html · photo.html 官网 GitHub

Material Icons

Google 提供的开源图标集，涵盖 2000+ 遵循 Material Design 规范的图标，本站通过 Google Fonts CSS 加载并在各页面中使用。

Apache 2.0 全站 图标库

Google VR View

Google 开源的 360° 全景浏览框架，可将全景图片和视频转化为交互式 VR 体验。该框架自 2018 年 12 月起停止维护，本站保留了基于此框架的旧版 VR 页面作为遗留资源。

Apache 2.0 vrview/ (Legacy) 官方文档 GitHub (Archived)
